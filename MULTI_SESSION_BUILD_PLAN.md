# Multi-Session Build Plan

## The Protocol

### Session Start Ritual
1. Read `CLINE_RULES.md`.
2. Read `CLINE_MEMORY.md`.
3. Read `CLINE_INDEX.md`.

### Session End Ritual
1. Update `CLINE_MEMORY.md`.
2. Update `CLINE_INDEX.md`.
3. Post Handoff Message.

### Handoff Template
```markdown
## Session Handoff
- **Phase:** [Current Phase]
- **Completed:** [List of completed items]
- **Blockers:** [Any blockers]
- **Next Step:** [What to do next]
```

## Build Phases
- **Phase A: Foundation** - Initialize project structure, rules, and registry.
- **Phase B: Core Architecture** - Setup TypeScript, Tailwind, and basic layouts.
- **Phase C: Feature Implementation** - Build out specific features.
- **Phase D: Testing & Optimization** - Refine code and add tests.
... (more phases as needed)

## Frozen Files
- `CLINE_RULES.md` (Unless specifically authorized to change)
- `MULTI_SESSION_BUILD_PLAN.md` (Unless evolving the process)
