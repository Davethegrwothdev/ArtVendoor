# Hero Artwork Placeholder

## Instructions

1. **Save your artwork image here**: Place the pointillist artwork image in this directory
2. **Rename the file**: Name it `hero-artwork.jpg` (or update the path in `index.html`)
3. **Optimize the image**: 
   - Recommended size: 1920x1080px or larger
   - Format: JPG (quality 80-85) or WebP
   - File size: Under 500KB for fast loading

## Image Requirements

- **Aspect Ratio**: 16:9 or wider
- **Minimum Resolution**: 1920x1080px
- **Format**: JPG, PNG, or WebP
- **File Name**: `hero-artwork.jpg`

## Optimization Tips

Use tools like:
- [TinyPNG](https://tinypng.com/) for PNG/JPG compression
- [Squoosh](https://squoosh.app/) for WebP conversion
- ImageOptim (Mac) or FileOptimizer (Windows)

## Alternative

If you want to use the artwork as a CSS background pattern instead, you can:
1. Create a smaller tileable version
2. Use it in the `.hero__background` element
3. Add a color overlay for text readability
